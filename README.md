# JobSphere 🚀

**JobSphere** is a modern and responsive **job board application** built using **React** and **Tailwind CSS**. It allows users to explore job opportunities, search and filter listings, save favorite jobs, and apply through an interactive modal—all using locally managed data.

---

## ✨ Features

- 🔍 **Search Jobs** by job title or company name  
- 🎯 **Filter Jobs** by job type (Full Time, Part Time, Contract, etc.)  
- 💾 **Save / Bookmark Jobs** for later viewing  
- 📝 **Apply Now Modal** with form submission  
- ✅ **Success Popup** after successful application  
- 📱 **Fully Responsive UI** (Mobile, Tablet & Desktop)  
- ⚡ **Fast & Lightweight** (No external API dependency)

---

## 🛠 Tech Stack

- **React (Vite / CRA)**
- **Tailwind CSS**
- **Lucide React Icons**
- **Local Data Management (JavaScript files)**
